El paràmetre d'entrada és el hash precalculat del document a signar (veure codi font).
La sortida és via event Javascript. Hi ha una funció Javascript (onMultiSignOK) que s'encarrega de capturar totes les signatures
generades (en el cas de signar múltiples documents) un cop s'han generat totes.
No s'ha definit ni el color de fons ni el logo, i per tant es mostren els valors per defecte (fons blanc i logo CATCert).
